# Recursive Fractal Drawing

A fractal visualizer built in Python using the Turtle graphics library. Demonstrates recursion through two fractal types — a binary tree and a Lévy C curve.

## Features

- Binary tree fractal with adjustable branch length and angle
- Lévy C curve fractal using recursive turn balancing
- Depth-first recursive traversal
- Base case depth limiting to control recursion budget
- Instant rendering with maximum turtle speed

## How to Run

```
python Turtle.py
```

Requires Python 3 and the Turtle graphics library, which is included in the Python standard library. No external dependencies.

## Concepts Demonstrated

- Recursion and call stack behavior
- Base case design
- Depth-first traversal
- Geometric transformation through cumulative rotation

## Built With

- Python
- Turtle graphics
